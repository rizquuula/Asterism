# Workspace Overview

The workspace controls runtime identity and configuration.

Default path: current directory (`.`) unless `WORKSPACE_DIR` is set.

## Important files

- `config.yaml` — app and provider settings
- `SOUL.md` — philosophy and principles
- `AGENT.md` — behavior and operational guide
- `PERSONALITY.md` — tone and communication style
- `mcp_servers/mcp_servers.json` — MCP server config
